# Fake News Detection & Generation

A Flask-based web application for detecting and generating fake news using machine learning.

## Features

- 🔍 Real-time fake news detection
- 🤖 AI-powered fake news generation
- ✏️ Real news modification simulation
- 📊 Batch processing for multiple articles
- 🎯 Confidence scores for predictions
